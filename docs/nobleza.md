## Sistema de Nobleza
### Propósito

El sistema de nobleza introduce **estatus político visible** sin permitir poder permanente ni plutocracia.
### Base del sistema

La nobleza se calcula **exclusivamente** sobre:
- veALEMTY activo
No sobre ALEM líquido.
### Jerarquía
- 👑 Reyes — Top 3 veALEMTY
- 🤴 Príncipes — Top 10–15
- 🏰 Duques — Top 50
Fuera del Top 50 no hay nobleza.
### NFTs de Nobleza
- No transferibles
- Se mintean y queman automáticamente
- Representan cargo político, no propiedad
### Visual pública

Formato oficial:
🏅 juanito.eth 👑 Rey
### Permisos
- Reyes: dirección estratégica, propuestas estructurales
- Príncipes: propuestas y voto
- Duques: voto y deliberación
### Decadencia

Al perder veALEMTY suficiente:
- se pierde la nobleza
- el NFT se quema
No existe poder heredado.
