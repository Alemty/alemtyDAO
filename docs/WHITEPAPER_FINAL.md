## ALEMty — Whitepaper Final
### Resumen

alemty es una **DAO social y económica** construida sobre **Base (L2 de Ethereum)** que separa explícitamente **mérito, valor y poder** para crear un sistema sostenible, gobernable y resistente a abuso.
No es un foro tokenizado ni un DEX con comunidad: es una **economía social programable**.
### Principios fundamentales
- El mérito no se compra.
- El valor puede circular.
- El poder no es líquido.
- La reputación no se borra, pero venderla tiene costo.
- La gobernanza requiere compromiso temporal.
Regla canónica:
El Dharma se gana.
El Aura se mueve.
El Karma se paga.
El nivel se respeta.
El poder se bloquea.
### Arquitectura general

alemty se compone de seis capas:
- Identidad (DID)
- Comunidad (social DAO)
- Economía (Aura / ALEM)
- Gobernanza (veALEMTY)
- Ejecución (DEX)
- Servicio (Consultora Web Espacial)
Cada capa está desacoplada para reducir riesgos y permitir evolución independiente.
