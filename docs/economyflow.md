
# Flujo Económico de ALEMty

## Vista general

El sistema económico sigue un flujo claro y unidireccional:

Participación → Dharma → Aura → ALEM → veALEMTY

---

## Flujo detallado

1. El usuario participa
2. Genera Dharma (mérito)
3. Cada Dharma genera Aura
4. Aura se usa o se convierte
5. Aura ↔ ALEM mediante pool
6. ALEM se vende o se bloquea
7. Bloqueo genera veALEMTY

---

## Puntos de fricción

- Postear cuesta Aura
- Karma bloquea XP
- Vender reputación genera Karma

---

## Pools

- Aura ↔ ALEM (interno)
- ALEM ↔ stable (DEX L2 Base)

---

## Resultado

- Valor nace de actividad real
- Poder nace de compromiso
- El sistema se autorregula
