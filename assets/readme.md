
La carpeta `assets/` contiene recursos estáticos compartidos por el frontend, incluyendo **datos**, **íconos** y **POAPs**. Aquí se almacenan archivos que no forman parte del código de la aplicación, pero que son necesarios para la presentación visual o para representar entidades externas como identidades, reconocimientos o metadata pública.

Los **icons** se utilizan para la UI (tokens, acciones, navegación, estados), mientras que los **POAPs** representan activos visuales vinculados a eventos, participación o logros dentro del ecosistema. Todo el contenido de esta carpeta es puramente estático, versionado junto al frontend y puede ser servido directamente desde IPFS como parte del bundle público.
